# Portfolio học phần - Lưu Tú Uyên

Website tĩnh HTML/CSS/JS cho học phần Công nghệ số & ứng dụng trí tuệ nhân tạo.

## Thông tin

- Sinh viên: Lưu Tú Uyên
- MSSV: 25071041
- Trường: Trường Quốc tế, Đại học Quốc gia Hà Nội
- Theme: International Digital Learning Portfolio, tone hồng pastel
- Số PDF: 6 file trong thư mục `files/`

## Ghi chú avatar

Ảnh cá nhân đang dùng tại `images/avatar.jpg`. File `images/avatar-placeholder.svg` được giữ lại như ảnh dự phòng nếu cần.

## Cấu trúc

- `index.html`: khung giao diện chính.
- `css/style.css`: theme, layout và responsive.
- `js/script.js`: dữ liệu hồ sơ, 6 case học tập và modal chi tiết.
- `files/`: PDF bài tập, đặt tên `tuan1.pdf` đến `tuan6.pdf`.
